# Contributing to Phenomenological Computational Dynamics (PCD)

Thank you for helping us scale the geometric language of hardware-software co-design! We welcome contributions to expand the PCD framework across new computational axes.

## Development Workflow
1. **Fork the Repository**: Create a personal copy of the repository on GitHub.
2. **Clone and Install**:
   ```bash
   git clone [https://github.com/PikkiHara/phenomenological_general_science.git](https://github.com/PikkiHara/phenomenological_general_science.git)
   cd phenomenological_general_science
   pip install -e .
   ```
3. **Create a Feature Branch**: Use descriptive naming (e.g., `feat/io-axis-manifold`).
4. **Implement Tests**: Ensure changes to the log-stabilized thermodynamics engine include active tensor validation scripts.
5. **Submit a Pull Request**: Provide a clear description of the phenomenological metrics mapped by your code changes.

## Code of Conduct
We are committed to fostering a collaborative, scientifically rigorous, and welcoming open-source community. Please treat all contributors with respect.
